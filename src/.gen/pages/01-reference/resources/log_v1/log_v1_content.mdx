## LogV1

### Available Operations

* [Get Logs For App](/log_v1/get_logs_for_app) - Returns a stream of logs for an [application](https://hathora.dev/docs/concepts/hathora-entities#application) using `appId`.
* [Get Logs For Process](/log_v1/get_logs_for_process) - Returns a stream of logs for a [process](https://hathora.dev/docs/concepts/hathora-entities#process) using `appId` and `processId`.
* [Get Logs For Deployment](/log_v1/get_logs_for_deployment) - Returns a stream of logs for a [deployment](https://hathora.dev/docs/concepts/hathora-entities#deployment) using `appId` and `deploymentId`.
