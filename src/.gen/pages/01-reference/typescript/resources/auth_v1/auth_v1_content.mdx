## Auth V1
Operations that allow you to generate a Hathora\-signed [JSON web token (JWT)](https://jwt.io/) for [player authentication](https://hathora.dev/docs/lobbies\-and\-matchmaking/auth\-service).

### Available Operations

* [Login Anonymous](/typescript/auth_v1/login_anonymous) - Returns a unique player token for an anonymous user.
* [Login Google](/typescript/auth_v1/login_google) - Returns a unique player token using a Google-signed OIDC `idToken`.
* [Login Nickname](/typescript/auth_v1/login_nickname) - Returns a unique player token with a specified nickname for a user.
