## AppV1

### Available Operations

* [CreateApp](/resources/appv1/create_app) - Create a new [application](https://hathora.dev/docs/concepts/hathora-entities#application).
* [DeleteApp](/resources/appv1/delete_app) - Delete an [application](https://hathora.dev/docs/concepts/hathora-entities#application) using `appId`. Your organization will lose access to this application.
* [GetAppInfo](/resources/appv1/get_app_info) - Get details for an [application](https://hathora.dev/docs/concepts/hathora-entities#application) using `appId`.
* [GetApps](/resources/appv1/get_apps) - Returns an unsorted list of your organizationâ€™s [applications](https://hathora.dev/docs/concepts/hathora-entities#application). An application is uniquely identified by an `appId`.
* [UpdateApp](/resources/appv1/update_app) - Update data for an existing [application](https://hathora.dev/docs/concepts/hathora-entities#application) using `appId`.
