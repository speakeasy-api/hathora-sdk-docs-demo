## DiscoveryV1

### Available Operations

* [GetPingServiceEndpoints](/resources/discovery_v1/get_ping_service_endpoints) - Returns an array of all regions with a host and port that a client can directly ping. Open a websocket connection to `wss://<host>:<port>/ws` and send a packet. To calculate ping, measure the time it takes to get an echo packet back.
