## AuthV1

### Available Operations

* [Login Anonymous](/auth_v1/login_anonymous) - Returns a unique player token for an anonymous user.
* [Login Nickname](/auth_v1/login_nickname) - Returns a unique player token with a specified nickname for a user.
* [Login Google](/auth_v1/login_google) - Returns a unique player token using a Google-signed OIDC `idToken`.
