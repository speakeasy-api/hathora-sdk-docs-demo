## AuthV1

### Available Operations

* [LoginAnonymous](/resources/authv1/login_anonymous) - Returns a unique player token for an anonymous user.
* [LoginGoogle](/resources/authv1/login_google) - Returns a unique player token using a Google-signed OIDC `idToken`.
* [LoginNickname](/resources/authv1/login_nickname) - Returns a unique player token with a specified nickname for a user.
