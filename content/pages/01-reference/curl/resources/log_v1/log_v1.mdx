## Log V1
Operations to get logs by [applications](https://hathora.dev/docs/concepts/hathora\-entities\#application), [processes](https://hathora.dev/docs/concepts/hathora\-entities\#process), and [deployments](https://hathora.dev/docs/concepts/hathora\-entities\#deployment). We store 20GB of logs data.

### Available Operations

* [Get Logs For App](/curl/log_v1/get_logs_for_app) - Returns a stream of logs for an [application](https://hathora.dev/docs/concepts/hathora-entities#application) using `appId`.
* [Get Logs For Process](/curl/log_v1/get_logs_for_process) - Returns a stream of logs for a [process](https://hathora.dev/docs/concepts/hathora-entities#process) using `appId` and `processId`.
* [Get Logs For Deployment](/curl/log_v1/get_logs_for_deployment) - Returns a stream of logs for a [deployment](https://hathora.dev/docs/concepts/hathora-entities#deployment) using `appId` and `deploymentId`.
